## Usage & Compliance

Synnoesis orchestrates Anthropic's Claude using **your own credentials**. It
bundles **no API keys** and grants no Claude access of its own.

- **Personal / single-user use:** a Claude subscription via the official Claude
  CLI is fine.
- **Serving other users or customers (multi-user / commercial):** you must use
  **your own Anthropic API key** under Anthropic's Commercial Terms — *not* a
  personal subscription. Routing other users through personal subscription
  credentials violates Anthropic's terms.
- You are responsible for your own compliance with Anthropic's
  [Consumer Terms, Commercial Terms, and Usage Policy](https://www.anthropic.com/legal).

**Bottom line:** use Synnoesis commercially all you like — just bring your own
keys, so you stay compliant and protected.
