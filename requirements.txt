# Synnoesis dependencies.
#
# The mesh FLOOR has NO required runtime dependencies — it runs on the Python
# standard library alone, so `pip install -r requirements.txt` installs nothing
# and can never fail to build.
#
# OPTIONAL — message signing. `cryptography` enables Ed25519 signing; without it
# the mesh runs in unsigned warn-mode. It is intentionally NOT a hard dependency,
# so the floor never fails to install. To enable signing:
#     python -m pip install --upgrade pip   # so pip uses the prebuilt wheel
#     pip install cryptography
